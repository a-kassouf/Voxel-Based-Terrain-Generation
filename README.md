# Voxel-Based-Terrain-Generation
A minecraft like game made with python and the Ursina library as well as the perlin noise library to showcase random terrain generation. 
Chunk rendering is also implemented so that the randomly generated terrain only appears or disappears based on your position. 

REQUIREMENTS ARE:

Having python installed
Having the Ursina library
Having the perlin noise library


CONTROLS:
1-2-3-4 to switch between blocks types 
right click to remove a block
left click to add it
W-A-S-D to move
space to jump
